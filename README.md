**Fleet Fuel Consumption & Trip Log Management System (JDBC)**

**Project Overview**
The Fleet Fuel Consumption & Trip Log Management System is a Java-based console application developed using JDBC and Oracle Database.

**This system helps manage:**
Vehicle records
Trip logs
Fuel purchase details
Fuel efficiency calculations

**It follows a layered architecture using:**
DAO Pattern
Service Layer
Bean Classes
Oracle SQL Database

**Technologies Used**
Java (Core Java)
JDBC
Oracle XE Database
SQL
DAO Design Pattern
Console-Based Application
